Developing social media application to connect users. The social media app is being developed with Microservices using both java and
javascript as background. The front end is being developing with React. 

Process to setup the project:
*, Clone the git project.
*, Use Spring Tool Sute to manage microservices.
*, Configure database in applications.properties.
*, Use Visual studio for Node js services.
*, Use visual studio or any other editor to view the react code (Front end).
*, Run both react and all micro services.



Technologies used : Springboot,Java,NodeJs,Javascipt,React,MongoDB and Mysql.
